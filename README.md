# GIF Collection

A clean, responsive GIF gallery with 260+ animated GIFs. Features modal viewer, slideshow, zoom, fullscreen, and dark/light themes.

## Quick Start

Open `index.html` in a browser or use a local server:
```bash
python -m http.server 8000
```

## Features

- 260+ curated GIFs
- Responsive grid (2-5 columns)
- Modal viewer with zoom & fullscreen
- Dark/light theme toggle
- Slideshow mode
- Keyboard shortcuts (press `?` for help)
- Download & share links

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `?` | Show help |
| `ESC` | Close |
| `←` `→` | Navigate |
| `F` | Fullscreen |
| `S` | Slideshow |
| `+` `-` | Zoom |

## Tech Stack

HTML5 • CSS3 • Vanilla JavaScript • Font Awesome
